Introduction:
This is a simple HTML and CSS project for a login page. It is a responsive design with a background image on the left side and a login form on the right side.

Files:

index.html: This file contains the HTML markup for the login page.
style.css: This file contains the CSS styling for the login page.
fonts/: This directory contains the custom font file used in the project.
images/: This directory contains the background image used in the project.
Getting Started:
To run the project, simply open the index.html file in a web browser. The login form should be displayed with the background image on the left.

Customization:
To customize the project, you can modify the CSS in the style.css file. You can change the font family, color scheme, background image, and more to fit your needs.

Media Queries:
This project is designed to be responsive using media queries. The CSS contains a media query for screen sizes smaller than 600px. This will adjust the layout and styling of the login form for mobile devices.

Dependencies:
This project has no dependencies and can be run without the need for any additional software or libraries.
